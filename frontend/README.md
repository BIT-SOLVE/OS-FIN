# UFOS Frontend

React TypeScript application for the Unified Financial Operating System.

## Technology Stack
- React
- TypeScript
- Vite
- CSS Modules

## Development Setup
1. `npm install`
2. `npm run dev`

## Configuration
Backend API URL is configured via environment variables in `.env` (or `.env.local`):
```
VITE_API_BASE_URL=http://localhost:8080/api/v1
```
